# Twitcord
A simple bot that helps you interact with Twitter in Discord.  

Twitcord is now open source! Vote it on [Top.gg](https://top.gg/bot/917122425102163971).  

## Build from source
### Build project
`cargo build --release`
### Build Docker image
`docker buildx build -t twitcord-rs .`
